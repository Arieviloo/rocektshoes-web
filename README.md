## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/Arieviloo/rocektshoes-web

# Go into the front path
$ cd rocektshoes-web

# Install dependencies
$ yarn install

# Run the Front
$ yarn start]

# Run the API(json-server)
$ json-server server.json -p 3333
```
